# ユースケース

二つのトラックは Knoll & van Dick (2013) の 4 形態サイレンス理論に対する補完的な研究問いに答える．

## Track A — 心理測定的再現

**目標**: 独立サンプルでも 4 因子相関 CFA モデル（`M3`）と Knoll 2013 の α / r プロファイルが再現するか？

**入力**: `data_external/`（gitignore；絶対に commit しない）に置く実調査 CSV．Prolific UK/US で収集（N ≈ 400）．任意で日本語サンプル（N ≈ 200）を文化間不変性検定に使用．

**出力**: `results/track_a/<sample>/` 配下:

- `loaded.csv` — クリーニング・スクリーニング済の調査行．
- `desc_stats.csv` — 項目別 M / SD / Skew / Kurt / 項目総和 r．
- `efa_loadings_<rotation>.csv`, `efa_eigenvalues_<rotation>.csv` — EFA 出力．
- `cfa_fit_indices.csv` — 競合モデル fit（M1 vs M2 vs M3 vs M3b vs M4）．
- `reliability.csv` — 下位尺度別 Cronbach α + Knoll target + Δ．
- `nomological_r_matrix.csv` — 4 × 6 Pearson r + 95% bootstrap CI + Knoll Table 2 との符号一致．
- `discriminant_validity.csv` — AVE / CR / 二乗 r の Fornell-Larcker 行列．
- `robustness_split_half.csv` — 200 回ランダム分割の split-half α 安定性．
- `multigroup_cfa_<samples>.csv` — 多群 CFA 不変性（configural / metric / scalar）．

**スモーク経路（実データなし）**: `knoll-tools survey-loader --synthesize-n 200 --sample synth` で合成データを `results/track_a/synth/loaded.csv` に書き出す．下流の全サブコマンドが消費可能．

## Track B — 生成的 ABM

**目標**: 4 動機 ABM が，rule と LLM の二つの mechanism 構成で，Knoll が観察したのと同じ動機別サイレンスパターンを生成できるか？

**入力**: CLI フラグのみ（`knoll run` / `knoll sweep`）．外部データ不要．

**Mode 1: rule（`--decision-mode rule`）**: 純粋 socsim コア．LLM 呼び出し 0 回．bit 決定論的．用途:

- 動機分布の決定論的ベースライン確立．
- β_ψ × β_f × β_ρ^{PS} × prosocial_decoupling のパラメータ空間 sweep．
- `--prosocial-climate-decoupling` トグルで PS-風土独立性知見を機構的に再現．

**Mode 2: LLM（`--decision-mode llm`）**: `socsim-llm`（Ollama → OpenAI フォールバック）が persona と 8 特徴量文脈を見てサイレンス動機を決定．用途:

- LLM ペルソナが β 制約なしでも質的な動機符号パターンを再現するかを検証．
- 同一 seed で LLM vs rule を比較し，LLM 層の寄与を切り分け．

**Mode 3: sweep**: デカルト積 sweep が `sweep_summary.csv` を出力．`visualize-sweep` がそれを応答曲線とヒートマップに変換（PS-風土相関 vs `β_ρ^{PS}`，β_ψ × β_f の動機 mix ヒートマップ，decoupling on/off での PS-風土 r 箱ひげ図）．

## 三者比較統合

Track A（実データ）と Track B の集団 CFA 創発を組み合わせ，`reproduce-paper` が "paper / Track A / Track B" の三列比較を出力する．
