# 可視化

## `knoll-tools visualize`

`run` ディレクトリ（既定 `results/latest`）を読んで同ディレクトリ（または `--output-dir`）に PNG を 3 枚出力する:

- **motive_mix_timeseries.png** — サイレンス集団内の各動機（AS / QS / PS / OS）の比率時系列．climate-of-silence の軌跡 C(t) を副軸に重ねる．rule / LLM ダイナミクスが時間方向の *質的な* 動機シフトを生むかを見る主プロット．
- **silence_kl_timeseries.png** — 1 ステップあたりのサイレンス率（左）と KL(π_emp || π_abm)（右）．KL → 0 は ABM の動機分布が Knoll Study 2 周辺値（AS=.22, QS=.27, PS=.40, OS=.18）と一致することを示す．
- **motive_climate_bar.png** — 最終ステップにおける各動機のサイレンスフラグベクトルとチーム climate-of-silence proxy の Pearson r．Knoll Table 2 の論文 target（.65 / .40 / .11 / .35）を並べて表示．**PS の棒が critical test** — `--prosocial-climate-decoupling` 時には 0.11 付近（信頼区間が 0 を含む）に位置するはず．

## `knoll-tools visualize-sweep`

`sweep` ディレクトリの `sweep_summary.csv` を読んで出力:

- **sweep_corr_ps_climate.png** — β_ρ^{PS} を横軸とした corr(PS, climate_of_silence)．`prosocial_climate_decoupling ∈ {False, True}` で別系列のエラーバー．Knoll の target r = .11（n.s.）を破線で表示．"decoupling=True" 系列は全 β_ρ^{PS} で 0 付近に張りつくべき．
- **sweep_motive_heatmap.png** — β_ψ × β_f セルでの mean `motive_mix_AS` ヒートマップ（seed と decoupling を平均）．心理的安全性 / 恐怖の勾配が AS 比率をどう動かすかを可視化．
- **sweep_decoupling_effect.png** — decoupling = False vs True での corr(PS, climate_of_silence) 箱ひげ．decoupling スイッチで PS-風土相関の CI が 0 を跨ぐ知見を可視化．

## `knoll-tools show-experiment-settings`

`config.json`（run）または `sweep_config.json`（sweep）と `run_metadata.json`（LLM 由来情報 + cache-hit 率）を整形表示．`--json` で JSON 出力に切替．
