# Use cases

The two tracks address complementary research questions about Knoll & van Dick's 2013 4-form silence theory.

## Track A — psychometric replication

**Goal.** Does the 4-factor correlated CFA model (`M3`) and the α / r profile from Knoll 2013 survive in an independent sample?

**Inputs.** A real survey CSV in `data_external/` (gitignored — never commit) collected via Prolific UK/US (N ≈ 400). Optionally a Japanese sample (N ≈ 200) for cross-cultural invariance.

**Outputs.** Under `results/track_a/<sample>/`:

- `loaded.csv` — cleaned + screened survey rows.
- `desc_stats.csv` — per-item M / SD / Skew / Kurt / item-total r.
- `efa_loadings_<rotation>.csv`, `efa_eigenvalues_<rotation>.csv` — EFA outputs.
- `cfa_fit_indices.csv` — competing-model fit (M1 vs M2 vs M3 vs M3b vs M4).
- `reliability.csv` — per-subscale Cronbach α + Knoll target + Δ.
- `nomological_r_matrix.csv` — 4 × 6 Pearson r + 95% bootstrap CI + sign-match against Knoll Table 2.
- `discriminant_validity.csv` — AVE / CR / squared-r Fornell-Larcker matrix.
- `robustness_split_half.csv` — split-half α stability across 200 random splits.
- `multigroup_cfa_<samples>.csv` — multigroup CFA invariance (configural / metric / scalar).

**Smoke path (no real data).** `knoll-tools survey-loader --synthesize-n 200 --sample synth` writes a synthetic dataset to `results/track_a/synth/loaded.csv` that every downstream subcommand can consume.

## Track B — generative ABM

**Goal.** Can a 4-motive ABM, with two mechanism configurations (rule vs LLM), generate the same motive-stratified silence patterns Knoll observed?

**Inputs.** CLI flags only (`knoll run` / `knoll sweep`). No external data.

**Mode 1: rule (`--decision-mode rule`).** Pure socsim core, zero LLM calls, bit-deterministic. Use to:

- Establish a deterministic baseline for the motive distribution.
- Sweep β_ψ × β_f × β_ρ^{PS} × prosocial_decoupling to map the parameter space.
- Mechanise the PS-climate independence finding by toggling `--prosocial-climate-decoupling`.

**Mode 2: LLM (`--decision-mode llm`).** `socsim-llm` (Ollama → OpenAI fallback) decides each silence motive given a persona + 8-feature context. Use to:

- Test whether LLM personas reproduce the qualitative motive sign pattern even without explicit β constraints.
- Compare LLM vs rule outputs at matched seeds to isolate the LLM layer's contribution.

**Mode 3: sweep.** Cartesian sweep produces a `sweep_summary.csv` that `visualize-sweep` turns into response curves and heatmaps (PS-climate correlation vs `β_ρ^{PS}`, motive-mix heatmaps across β_ψ × β_f, boxplots of PS-climate r under decoupling on/off).

## 3-way integration

Combining Track A (real data) with Track B population-CFA emergence, `reproduce-paper` produces a 3-column "paper / Track A / Track B" comparison.
