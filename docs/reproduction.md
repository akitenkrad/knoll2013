## What you can do right now

1. **Run rule-mode** and inspect the motive-mix time series:
   ```bash
   cargo run --release -- run --decision-mode rule \
       --n-teams 8 --team-size 12 \
       --prosocial-climate-decoupling --t-max 36 --seed 42
   uv run knoll-tools visualize
   ```
   Look at `motive_mix_timeseries.png`. Does the PS share converge to ≈ 0.40 (Knoll Study 2 marginal)?

2. **Run the β sweep** to inspect PS-climate independence:
   ```bash
   cargo run --release -- sweep --beta-rho-ps-values "0.0,0.1,0.3" --runs 10 --seed 42
   uv run knoll-tools visualize-sweep
   ```
   Look at `sweep_corr_ps_climate.png`. With `decoupling=True`, the PS-climate r should track the Knoll target line of .11 across all β_ρ^{PS} values.

3. **Run LLM mode** (Ollama required):
   ```bash
   ollama pull llama3.1
   export OLLAMA_HOST=http://localhost:11434 OLLAMA_MODEL=llama3.1
   cargo run --release -- run --decision-mode llm --llm-cache-path runs/knoll_cache.json --t-max 24 --seed 42
   ```
   On a warm cache, re-runs are nearly instant and bit-identical.

4. **Track A synthetic-data sanity check**:
   ```bash
   uv run knoll-tools survey-loader --synthesize-n 200 --sample synth
   uv run knoll-tools reliability-analysis --sample synth
   uv run knoll-tools cfa-competing-models --sample synth --models M1,M2,M3,M3b,M4
   uv run knoll-tools nomological-network --sample synth --bootstrap 500
   ```
   On synthetic data calibrated to Knoll Study 2 marginals, you should see α ≈ .80–.86 per subscale, M3 (correlated 4-factor) winning over M1 / M2 by ΔCFI ≥ .15, and a > 90% sign-match rate vs Knoll Table 2 paper values.

## What is deferred

- **Phase B3** — emit reflexive 12-item self-ratings (`knoll reproduce --emit-self-ratings`); fit population-CFA via semopy in `knoll-tools cfa-analysis`; compare χ² / CFI / RMSEA / α with Knoll Study 1 targets.
- **Phase X** — 3-way "paper / Track A / Track B" comparison via `knoll-tools reproduce-paper`. Prerequisites: Track A real data + Phase B3.
- **Phase A1–A5** — actual survey instrument finalisation, IRB approval, data collection, multigroup invariance. The `survey/` directory carries placeholders only.
