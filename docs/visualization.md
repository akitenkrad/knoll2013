# Visualization

## `knoll-tools visualize`

Reads a `run` directory (default `results/latest`) and produces three PNGs in that same directory (or in `--output-dir`):

- **motive_mix_timeseries.png** — per-step share of each motive (AS / QS / PS / OS) within the silent population, overlaid with the climate-of-silence trajectory C(t) on a secondary y-axis. The headline plot for inspecting whether the rule / LLM dynamics produce a *qualitative* motive shift over time.
- **silence_kl_timeseries.png** — silence rate per step (left) + KL(π_emp || π_abm) per step (right). KL → 0 means the ABM motive distribution matches Knoll Study 2 marginals (AS=.22, QS=.27, PS=.40, OS=.18).
- **motive_climate_bar.png** — final-step Pearson r between each motive's silence-flag vector and the team's climate-of-silence proxy, side-by-side with the Knoll Table 2 paper targets (.65 / .40 / .11 / .35). The PS bar is the critical test — it should sit near 0.11 (and overlap 0 in CI) when `--prosocial-climate-decoupling` is set.

## `knoll-tools visualize-sweep`

Reads a `sweep` directory's `sweep_summary.csv` and produces:

- **sweep_corr_ps_climate.png** — corr(PS, climate_of_silence) as a function of β_ρ^{PS}, with separate error-bar lines for `prosocial_climate_decoupling ∈ {False, True}`. The Knoll target r = .11 (n.s.) is shown as a dashed horizontal line. The "decoupling=True" line should hover near 0 across all β_ρ^{PS} values.
- **sweep_motive_heatmap.png** — mean `motive_mix_AS` heatmap across β_ψ × β_f cells (averaged over seeds + decoupling). Heat shows how psychological-safety / fear gradients shift the AS share.
- **sweep_decoupling_effect.png** — boxplot of corr(PS, climate_of_silence) under decoupling = False vs True. Visualises the headline result that the decoupling switch produces a CI-overlapping-0 PS-climate correlation.

## `knoll-tools show-experiment-settings`

Pretty-prints `config.json` (run) or `sweep_config.json` (sweep) plus `run_metadata.json` (LLM provenance + cache-hit rate). Use `--json` to emit JSON instead of the table form.
